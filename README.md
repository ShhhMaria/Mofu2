# PawPrint Pet Planner

A cute and aesthetic pet planner app to manage your pets, track daily tasks, and generate custom care schedules.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Run the app:
   ```
   npm run dev
   ```

## Features

- Manage multiple pets with detailed information
- Create and track daily care tasks
- Generate care schedules
- User profile and authentication
- Responsive design for mobile and desktop
